Рабочая страница метеорной группы LSME

Здесь собраны ссылки на программное обеспечение, базы данных и активные метеорные сети.

Программное обеспечение для распознавания метеоров

  • UFOCaptureV2 – Windows
  • MetRec – Windows
  • GMN software – Linux
  • Freeture - Открытое ПО французской сети FRIPON (Detection under Windows)
  • Fast Meteor Detection Toolbox – используется Жереми Ваюбаон
  • MetDetPy
  • MeteorDetector

Доступные базы данных

  • Основная база IAU MDC – каталоги CAMS, SonotaCo, EDMOND, GMN и др.
  • SonotaCo – японская сеть, ежегодные каталоги
  • EDMOND – Европейская база, ежегодные обновления
  • FRIPON – сеть на базе IMCCE, Observatoire de Paris, (данные до 2022 г. ?)
  • CNEOS Fireballs – данные о ярких метеорах (без орбит)
  • GOES Bolides – данные с геостационарных спутников GOES East и GOES West, ежедневные обновления с задержкой 2 недели
  • GMN data – ежедневные обновления

Работающие метеорные сети

  • CAMS – радианты и визуализация (FDL и meteorshowers), но без аозможности скачивания, ПО только для участников сети
  • NASA’s All Sky Fireball Network – Доступны данные за последние 3 недели (орбиты, графики, видео)
  • SonotaCo – японская сеть, ПО: UFOCapture, UFOAnalyzer, UFOOrbit
  • Western Ontario – радарные и оптические наблюдения университета Заподного Онтарио, ПО ASGARD (закрытое)
  • EDMOND – европейская видеосеть eMetN: EDMOND Использует ПО UFO...
  • FRIPON – европейская сеть на базе IMCCE, ПО: Freeture
  • GOES satellites (GLM) – спутниковые регистрации болидов. Свое ПО на основе нейронных сетей.
  • GMN – Global Meteor Network
