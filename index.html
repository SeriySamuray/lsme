Рабочая страница научной группы по метеорным исследованиям

Здесь собраны ссылки на используемое программное обеспечение, базы данных и активные метеорные сети.

Программное обеспечение для распознавания метеоров в видеопотоке

  • UFOCaptureV2 – Windows
  • MetRec – Windows
  • GMN software
  • Fast Meteor Detection Toolbox – используется Жереми Ваюбаон
  • MetDetPy
  • MeteorDetector

Доступные базы данных

  • Основная база IAU MDC – каталоги CAMS, Sonotaco, EDMOND, GMN и др.
  • SonotaCo – японская сеть, ежегодные каталоги
  • EDMOND – Европейская база, ежегодные обновления
  • FRIPON – сеть на базе IMCCE, Observatoire de Paris
  • CNEOS Fireballs – данные о ярких метеорах (без орбит)
  • GOES Bolides – данные с геостационарных спутников GOES
  • GMN data – свежие орбитальные данные

Работающие метеорные сети

  • CAMS – радианты и визуализация, данные только для участников
  • NASA’s All Sky Fireball Network – последние 3 недели (орбиты, графики, видео)
  • SonotaCo – японская сеть, ПО: UFOCapture, UFOAnalyzer, UFOOrbit
  • Western Ontario – радарные и оптические наблюдения, ПО ASGARD (закрытое)
  • EDMOND – европейская видеосеть
  • FRIPON – европейская сеть, ПО: Freeture
  • GOES satellites (GLM) – спутниковые регистрации болидов
  • GMN – Global Meteor Network
